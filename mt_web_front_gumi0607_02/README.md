###### 사용자는 회원 가입을 통해서 시스템을 사용할 수 있다. 회원 가입 시는 아이디와 이름,비밀번호를 반드시 제공해야 하며 이 항목들이 입력되지 않으면 가입할 수 없다.

# F07-회원 정보 조회
###### 로그인한 사용자의 상세 정보를 통해 사용자의 아이디와 주문 내역을 확인할 수 있다.
![7](/uploads/f45c811522cbe6a7156e42af76ee7339/7.jpg)

# F08-주문 상세 조회
###### 주문 번호를 통해 주문의 상세 정보를 조회할 수 있다. 주문 상세에는 이전에 주문한 물품의 정보와 총 비용, 주문을 통해서 적립한 스탬프의 개수정보가 출력된다.
![8](/uploads/5839c77e39b29f129ca3f6da90ded0fb/8.jpg)

# F09-상품별 정보 조회
###### 상품별로 상품의 상세 정보를 출력한다. 여기에는 상품이름, 상품 이미지, 단가와 함께 총 주문 수량을 출력해서 인기도를 측정할 수 있게 한다. 평가 회수, 평균 평점, 한 줄 평 관련 내용은 추가 기능이다.

# F11-상품별 상품평가 조회
###### 상품별로 등록한 상품 평 정보를 출력한다. 이때 로그인한 사용자가 작성한 상품평에 대해서만 수정, 삭제 버튼이 보이고 다른 사용자가 작성한 내용은 단지 확인만 가능하다. 상품에 대한 상품 평가 회수 및 평균 평점이 상품 정보와 함께 표현된다.
![11](/uploads/df2d6cdd2dbae398d1d34fb3d2f35a46/11.jpg)

![12](/uploads/ab4ae4d463676b02056f5dcce6b072e5/12.jpg)
